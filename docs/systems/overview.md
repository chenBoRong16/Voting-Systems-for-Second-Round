# 選制說明總覽

這一頁是「第二輪選制測試」所支援選制的入口：點選選制名稱可看到直覺、計票步驟、優缺點與例子。

如果你只想先抓住大方向：先看下面的「快速對照表」；再按需求點進各選制細節。

## 快速對照（給大眾與專家）

| 選制 | 主要用到的票資訊 | 是否符合孔多賽贏家標準 | 一句話直覺 |
|---|---|---|---|
| [簡單多數（只看第一名）](plurality.md) | 只看第一名 | 否 | 誰第一名票最多誰贏 |
| [IRV（即時決選 / 排序投票）](irv.md) | 完整排序 | 否 | 一輪輪淘汰最低第一名，把票轉移 |
| [Borda（波達計分）](borda.md) | 完整排序 | 否 | 依名次給分，加總最高者勝 |
| [Ranked Pairs（鎖定排序）](ranked-pairs.md) | 完整排序 → 兩兩對決 | ✅ 是 | 依勝負強度鎖定勝負關係，避免循環 |
| [Benham](benham.md) | 完整排序 → 兩兩對決 + 第一名淘汰 | ✅ 是 | 先找兩兩都能贏的人；否則像 IRV 淘汰 |
| [Minimax](minimax.md) | 完整排序 → 兩兩對決 | ✅ 是 | 看每個人的「最糟對決」誰最不糟 |

> ✅「符合孔多賽贏家標準」（也常說「孔多賽一致 / Condorcet-consistent」）：
> 若存在孔多賽優勝者（在兩兩對決中能擊敗所有其他候選人），該選制一定會選出他。

**本頁要特別註明的 3 個孔多賽系列選制**：Ranked Pairs、Benham、Minimax（以上三者皆 ✅ 符合孔多賽贏家標準）。

## 這些選制在第二輪的角色

若你採用「Top4 兩輪」：

- 第一輪常用簡單多數（每張票只選 1 位，不排序）做候選人篩選：規則直觀、容易操作
- 第二輪使用「完整排序票」的選制決定勝者：每張票都把候選人從第一名排到最後一名，更完整反映偏好
- 按本專案假設：若總候選人數 ≤ 1，視為無需辦理選舉；若總候選人數為 2～4，第一輪不會淘汰任何人，因此可直接跳過第一輪、直接進第二輪
- （補充）即使採用 Top4，第二輪仍可能因退選、資格等原因少於 4 人
- 第一輪的目的，是在人數過多時先縮減候選人數，避免選票過長、難以人工計票而被迫集中或改成電子計票，進而增加對選制的不信任

一句話理解「Top4 兩輪」：
第一輪先把人選縮到 4 人，第二輪再用完整排序票來決定最終勝者。
