# Borda（波達計分）

## 一句話概念

把名次轉成分數：越高排名分數越高，總分最高者勝。

## 計票步驟（本網站的詳細輸出）

若有 N 位候選人：

1. 每張票：第 1 名得 N-1 分、第 2 名得 N-2 分……第 N 名得 0 分
2. 本網站會列出每位候選人的「名次分布」與加總：
   - 例如：`1名×票數×(N-1)分，2名×票數×(N-2)分，…`
   - 然後加總成該候選人的總分
3. 總分最高者當選

## 優點

- 充分利用排序資訊
- 不需要多輪淘汰，計算直接

## 缺點與注意

- 可能不符合孔多賽準則
- 對「中間偏好」特別敏感，可能讓大家普遍覺得還可以但不熱愛的人勝出

## 在第二輪的解讀

Borda 在第二輪常被拿來做對照：

- 若某人常被排在第 2 名，也可能靠積分勝出

## 例子 1（3 人）

投票分布：

- 5 票：A > B > C
- 4 票：B > C > A
- 1 票：C > A > B

以 3 人為例：第 1 名 2 分、第 2 名 1 分、第 3 名 0 分。  
A：5×2 + 4×0 + 1×1 = 11  
B：5×1 + 4×2 + 1×0 = 13  
C：5×0 + 4×1 + 1×2 = 6  
因此 **B 總分最高**。

## 例子 2（4 人）

投票分布：

- 6 票：A > B > C > D
- 4 票：B > C > D > A
- 3 票：C > D > B > A
- 2 票：D > C > B > A

以 4 人為例：第 1 名 3 分、第 2 名 2 分、第 3 名 1 分、第 4 名 0 分。  
A：6×3 + 4×0 + 3×0 + 2×0 = 18  
B：6×2 + 4×3 + 3×1 + 2×1 = 17  
C：6×1 + 4×2 + 3×3 + 2×2 = 27  
D：6×0 + 4×1 + 3×2 + 2×3 = 16  
因此 **C 總分最高**。

## 例子 3（4 人）

投票分布：

- 7 票：A > B > C > D
- 3 票：B > C > D > A
- 2 票：C > D > B > A
- 1 票：D > C > B > A

A：7×3 + 3×0 + 2×0 + 1×0 = 21  
B：7×2 + 3×3 + 2×1 + 1×1 = 26  
C：7×1 + 3×2 + 2×3 + 1×2 = 21  
D：7×0 + 3×1 + 2×2 + 1×3 = 10  
因此 **B 總分最高**。
